# Sheetforge — Portfolio Project

# Overview

A modern web application focused on:

* Excel conversion
* Document reading
* Image OCR
* Clean data extraction
* Fast file processing

The platform is designed as a lightweight SaaS-style portfolio project with a modern UI and smooth workflow.

---

# Main Features

## Sheetforge

### Supported Files

* XLSX
* XLS
* CSV

### Features

* Upload Excel files
* Preview spreadsheet data
* Convert files to CSV or JSON
* Download converted files
* Drag and drop upload

---

# Document Reader

### Supported Files

* PDF
* DOCX
* TXT

### Features

* Extract text from documents
* Read tables from PDFs
* Preview parsed content
* Export extracted data

---

# Image Reader (OCR)

### Supported Files

* PNG
* JPG
* JPEG

### Features

* OCR text extraction
* Detect tables from images
* Convert image tables into Excel
* Clean extracted text
